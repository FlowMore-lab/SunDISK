# SunDISK

**SunDISK** is a lightweight FTP client with a graphical user interface, developed specifically for working with the SunFile cloud storage. The application provides convenient access to files on the server via the FTP protocol.

## Features

- **File Manager** — browse, upload, download, delete files and folders
- **Built-in Image Viewer** — zoom, fullscreen mode
- **Text File Preview** — read TXT files without downloading
- **Media Support** — open audio and video files in system player
- **Dark Theme** — modern dark interface design
- **Portable** — single EXE file, no installation required
- **Multi-User Support** — works with any SunFile account

## Tech Stack

| Component | Technology |
|-----------|------------|
| GUI Framework | PyQt5 |
| FTP Protocol | ftplib (built-in) |
| Build Tool | PyInstaller |
| Platform | Windows 7/8/10/11 |

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Double-click file | Preview file |
| Double-click folder | Open folder |
| Backspace | Go to parent folder |
| F5 | Refresh file list |
| Delete | Delete selected file |
